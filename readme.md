# Scene Search

## What is this
Scene Search is a Backend built for storing Prefabs and tools for VRChat Creators with discoverability and Searchability at the core of its development. This is an attempted to serve as a better platform with advanced context aware searching to offer results that are relevent to the users in your service. This has started as a alternative for the VRCPrefabs database and should server as a masive upgrade for creators.
